## ūü§Ė Telegram Bot Setup

1. Talk to [@BotFather](https://t.me/BotFather) on Telegram ‚Üí `/newbot`
2. Copy your bot token
3. Send `/start` to your bot, then visit:
   `https://api.telegram.org/bot<TOKEN>/getUpdates`
4. Find your `chat_id`
5. In Nexus ‚Üí **Telegram Bot Panel** ‚Üí option **1** ‚Üí enter token & chat ID
6. Start polling ‚Üí receive live server stats on Telegram
